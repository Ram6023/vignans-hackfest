# Contributing to Hackify

Thank you for your interest in contributing to Hackify! As an open-source project, we welcome contributions from everyone.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please open an issue and include as much detail as possible, including steps to reproduce.

### Suggesting Enhancements
We are always looking for ways to improve the Innovation Portal. Feel free to suggest new features or UI improvements.

### Pull Requests
1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Commit your changes with descriptive commit messages.
4. Push to your fork and submit a pull request.

## Local Development
- `npm install` to install dependencies.
- `npm run dev` to start the development server.
